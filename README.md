# Starting the Event Pro app:

---

### 1. Open three terminals.

### 2. In terminal 1 run:

```
npm i
npm run dev
```

### 3. In terminal 2 run:

```
npm run serve
```

### 4. In terminal 3 run:

```
cd socket.io-server
npm run serve:socket
```

# Reseeding the database:

---

### Open a new terminal and run:

```
node reseed.js
```
